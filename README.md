## 🎯 Objective
To ensure the reliability, functionality, and usability of the CraftersLab platform by performing structured manual testing based on SRS requirements and real-world user behavior.

## 📌 Scope of Testing
*   **Functional Testing:** Validating core features against business requirements.
*   **UI/UX Validation:** Ensuring design consistency and intuitive user flow.
*   **Cross-Browser Compatibility:** Testing across Chrome, Firefox, and Edge.
*   **Regression Testing:** Verifying stability after new builds.
*   **Negative Testing:** Validating system resilience against invalid inputs.

---

## 📄 QA Deliverables
This repository contains professionally organized testing artifacts:
1.  **SRS Analysis:** Requirement breakdown and test scenario mapping.
2.  **Test Plan:** Defining strategy, environment, and entry/exit criteria.
3.  **Bug Reports:** Detailed defect logs with reproduction steps, severity, and priority.
4.  **Execution Reports:** Summary of pass/fail status and final quality sign-off.
