# CraftersLab Web & Extension QA Project

This repository contains end-to-end Quality Assurance (QA) documentation and testing work for the CraftersLab web application and browser extension.

## 📌 Project Overview
The goal of this project is to ensure the quality, reliability, and usability of the CraftersLab platform through structured testing activities.

## 🧪 What’s Included
- 📄 Software Requirements Specification (SRS) analysis
- 📝 Test Plan documentation
- ✅ Test Case design and execution
- 🐞 Bug Reports and tracking
- 🌐 Web application testing
- 🧩 Browser extension testing

## 🎯 Scope
- Functional Testing  
- UI/UX Testing  
- Cross-browser Testing  
- Compatibility Testing  
- Basic Regression Testing  

## 🛠 Tools Used
- Manual Testing
- Google Chrome DevTools
- Jira / Bug tracking 
  Microsoft Word for bug reporting

## 📊 Outcome
Improved product quality through systematic identification of issues and validation against requirements.
